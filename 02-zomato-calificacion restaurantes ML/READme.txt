# Predicción de Calificaciones de Restaurantes (Zomato Dataset)

Este proyecto aplica técnicas de aprendizaje automático supervisado utilizando Python para predecir la calificación de restaurantes. Se trabajó con un conjunto de datos proporcionado por Zomato, una plataforma líder en pedidos y entrega de comida.

## 🧠 Contexto
Zomato ha acumulado una gran cantidad de datos sobre restaurantes, tipos de cocina, ubicaciones, precios, servicios y calificaciones de usuarios. Este conjunto de datos se convierte en una fuente valiosa para analizar y entender los factores que influyen en la satisfacción del cliente.

## 🎯 Objetivos
- Usar técnicas de regresión supervisada para predecir calificaciones promedio de restaurantes.
- Evaluar el rendimiento de los modelos usando métricas como el error cuadrático medio (MSE).
- Visualizar relaciones entre variables e interpretar los resultados.

## 📋 Resumen del proyecto
El proyecto incluye:
- Limpieza y exploración del dataset de Zomato.
- Selección de características relevantes para la predicción.
- Implementación de un modelo de regresión lineal.
- Evaluación del modelo con métricas cuantitativas.
- Visualización de resultados y conclusiones.

## 🧰 Tecnologías y herramientas utilizadas
- Python 3.x
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
