# 📂 Conexión y Manejo de SQLite en Python

Este repositorio contiene un **Jupyter Notebook** (`sqlitelimpio.ipynb`) que explica paso a paso cómo trabajar con una base de datos **SQLite** usando Python.  
El objetivo es mostrar de manera clara cómo conectar, crear tablas, insertar datos y realizar consultas SQL, integrando los resultados con **pandas** para su análisis.

---

## 📂 Contenido

- `sqlitelimpio.ipynb` → Notebook con ejemplos prácticos de conexión y uso de SQLite en Python.

---

## ⚙️ Requisitos

Para ejecutar el notebook necesitas:

- [Python 3.8+](https://www.python.org/downloads/)
- [Jupyter Notebook](https://jupyter.org/install)
- Librerías:
  - `sqlite3` (incluida en la librería estándar de Python)
  - `pandas` (para análisis de datos)
  - `matplotlib` (opcional, para visualizaciones)

Instala dependencias opcionales con:

```bash
pip install pandas matplotlib
