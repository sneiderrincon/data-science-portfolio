# 📊 Financial Analysis Dashboard

This repository contains a **Power BI project** for financial data analysis, supported by a structured dataset in Excel.  
The project aims to provide insights into financial performance, cost analysis, and decision-making through interactive visualizations.

---

## 📂 Repository Contents
- **`analisis financiero.pbix`** → Power BI dashboard with interactive reports and visualizations.  
- **`Base Financiero.xlsx`** → Financial dataset used as the source for the dashboard.

---

## 🚀 Features
- Revenue and expense tracking.
- Profitability and margin analysis.
- Interactive KPIs with drill-down capabilities.
- Trend analysis over time.
- Visualizations for decision-making support.

---

## 🛠️ Tools & Technologies
- **Power BI** → Dashboard design and data visualization.  
- **Excel (XLSX)** → Raw financial dataset.  

---

## 📌 How to Use
1. Download or clone this repository.  
2. Open `analisis financiero.pbix` with **Power BI Desktop**.  
3. Ensure that `Base Financiero.xlsx` is located in the same folder as the `.pbix` file (or update the data source path inside Power BI).  
4. Refresh the dataset to explore the interactive dashboard.
