# 📊 Financial Summary Dashboard (SRC)

This repository contains a **Power BI project** focused on financial and sales analysis, using an Excel dataset as its source.  
The dashboard provides a clear and interactive overview of key financial indicators, helping to monitor performance and support decision-making.

---

## 📂 Repository Contents
- **`RESUMEN FINANCIERO SRC.pbix`** → Power BI dashboard with financial summary and sales insights.  
- **`Ventas.xlsx`** → Sales and financial dataset used as the data source for the dashboard.  

---

## 🚀 Features
- Sales performance analysis by period.  
- Revenue and profitability monitoring.  
- KPIs with interactive filters and drill-down capabilities.  
- Trend visualization for financial forecasting.  
- Comparative analysis of different financial indicators.  

---

## 🛠️ Tools & Technologies
- **Power BI** → Dashboard design and data visualization.  
- **Excel (XLSX)** → Structured dataset with sales and financial data.  

---

## 📌 How to Use
1. Download or clone this repository.  
2. Open `RESUMEN FINANCIERO SRC.pbix` with **Power BI Desktop**.  
3. Make sure `Ventas.xlsx` is in the same folder as the `.pbix` file (or update the data source path in Power BI).  
4. Refresh the dataset and start exploring the dashboard interactively.
